# WG Hope Bot v1.1.0

Релиз `v1.1.0` включает обновленный Telegram-бот и веб-мониторинг.

## Основные изменения

- Добавлен веб-мониторинг на FastAPI (`monitor/`) с авторизацией и настройками учетных данных.
- Добавлены графики трафика/скорости/handshake с периодами `1h`, `24h`, `7d`, `30d`, `all`.
- Добавлены фильтры и удобная навигация по событиям (поиск, тип событий, конфиг, режим "показать всё", пагинация).
- Добавлено логирование истории чатов в `chat/<chat_id>.log`.
- В админке бота добавлены:
  - раздел `Кастомизация`;
  - редактирование текстов `Инструкция` и `Вопросы / Поддержка` из интерфейса;
  - ссылка на мониторинг;
  - улучшенная структура кнопок и сценариев.
- Добавлена синхронизация профилей Telegram из админки.
- Улучшена работа с именами конфигов:
  - поддержка русских названий;
  - отображение без `chat_id`;
  - хранение `display_name`.

## Исправления

- Исправлена проблема, когда `Изменить поддержку` не срабатывало в админке.
- Исправлен ложный успех при установке лимита несуществующему `chat_id`.
- Исправлена логика фильтрации событий по конфигу: фильтрация выполняется на SQL-уровне до `LIMIT`.

## Обновления документации

- Обновлены `README.md`, `.env.example` и `CHANGELOG.md`.
