# WG Hope Bot v1.1.0

Релиз `v1.1.0` содержит крупное функциональное обновление бота и мониторинга.

## 🚀 Что нового

- 🌐 Добавили полноценную веб-панель мониторинга (`monitor/`) на FastAPI
- 📈 Появились графики по:
  - трафику
  - средней скорости
  - handshake
- 🕒 Добавили периоды просмотра: `1h`, `24h`, `7d`, `30d`, `all`
- 🔎 Добавили фильтры и удобную навигацию по событиям:
  - поиск
  - фильтр по типу события
  - фильтр по конфигу
  - режим "Показать всё" + пагинация
- 💬 Добавили историю чатов пользователей в `chat/<chat_id>.log`

## 🤖 Улучшения в Telegram-боте

- 🧭 Добавлен раздел `Кастомизация` в админке
- ✍️ Тексты `Инструкция` и `Вопросы / Поддержка` теперь редактируются прямо из админки
- 🔗 Добавлена кнопка с переходом в мониторинг
- 👤 Добавлена синхронизация Telegram-профилей (username/имя)
- 🧾 Улучшена работа с именами конфигов:
  - поддержка русских названий
  - отображение без `chat_id`
  - хранение человекочитаемого `display_name`

## 🧯 Исправления

- Исправили проблему с кнопкой `Изменить поддержку`
- Убрали ложный успех при установке лимита несуществующему `chat_id`
- Исправили фильтрацию событий по конфигу (теперь корректно работает до SQL `LIMIT`)
