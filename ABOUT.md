# 📘 About WG Hope Bot

WG Hope Bot - это Telegram-бот для управления доступом к личному WireGuard VPN-серверу без ручной выдачи конфигов.

## Зачем создан проект

Проект решает типовой сценарий: администратору нужно быстро выдавать и отзывать доступ к VPN пользователям, при этом сохранять порядок, журнал действий и контроль прав.

## Для кого

- Для владельцев личных и командных VPN-серверов
- Для небольших сообществ и команд с доступом по приглашениям
- Для администраторов, которым нужен простой интерфейс в Telegram

## Что делает бот

- Выдает клиентские WireGuard-конфиги и QR-коды
- Ведет роли пользователей и доступы
- Дает админ-панель для одобрения, блокировок и лимитов
- Хранит журнал событий и базовую статистику
- Работает с Docker и обычным запуском на Linux

## Как устроен процесс

1. Пользователь пишет `/start` и попадает в `pending`
2. Администратор подтверждает доступ
3. Пользователь создает конфиг через меню
4. Бот добавляет peer в WireGuard и отправляет `.conf` + QR
5. При отзыве бот удаляет peer и связанные клиентские файлы

## Что важно перед запуском

- Нужен установленный WireGuard (`wg`, `wg-quick`)
- Нужен доступ к серверному `wg0.conf`
- Нужны корректные переменные окружения (`.env`)
