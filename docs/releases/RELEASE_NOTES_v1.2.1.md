# WG Hope Bot v1.2.1

Patch-релиз с улучшением UX при создании нового конфига в Telegram-боте.

## Что изменено

- Шаг создания конфига стал короче и понятнее:
  - было: `Введите название конфига (латиница/цифры, пробелы заменятся на _). Для отмены нажмите 'Назад'.`
  - стало: `Введи название конфига`
- На шаге ввода названия теперь показывается отдельная клавиатура с кнопкой `Назад`.
- Нажатие `Назад` корректно отменяет создание и возвращает пользователя в главное меню.
