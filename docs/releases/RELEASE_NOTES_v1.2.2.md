# WG Hope Bot v1.2.2

Релиз нацелен на стабильность маршрутизации и улучшение рассылок в админке Telegram-бота.

## Что изменено

- Рассылки в админке переработаны в пошаговый flow:
  - группы получателей: `Ожидают`, `Одобренные`, `Забаненные`, `Все`;
  - отдельное добавление `chat_id` вручную поверх выбранной группы;
  - явный переход к шагу ввода контента.

- Рассылки теперь поддерживают не только текст:
  - `text`, `photo`, `video`, `document` (включая подписи для медиа/документов).

- Отчёт о рассылке стал подробным:
  - по неуспешным отправкам показываются `chat_id`, `@username` и причина ошибки.

- Исправлена критичная проблема с “воскрешением” удалённых сущностей после рестарта:
  - дефолтные `uplink_interfaces`/`uplink_regions` теперь сидируются только на пустой БД;
  - удалённые интерфейсы/регионы больше не добавляются обратно автоматически.

- Улучшена устойчивость синхронизации маршрутизации:
  - перед установкой актуальных правил выполняется очистка устаревших managed-правил `iptables`.

- Улучшена самовосстановимость конфигурации регионов:
  - если `default`-регион отсутствует, система назначает корректный `default` автоматически.

- Устранён warning при старте бота о порядке импорта/исполнения модуля `vpn_bot.main`.
