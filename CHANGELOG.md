# Changelog

## v1.0.0

- Полный рефакторинг архитектуры бота на модульную структуру `vpn_bot/*`
- Добавлена ролевая модель: `super_owner`, `admin`, `user`, `pending`, `banned`
- Добавлены админ-инструменты: управление пользователями, рассылки, лимиты, логи, статистика
- Реализовано хранение данных в SQLite с миграцией схемы
- Улучшено управление клиентскими конфигами и файлами (`.conf`, `.png`, `.json`, ключи)
- Обновлены `Dockerfile` и `docker-compose.yml` под новую структуру
- Обновлена документация и шаблон окружения
- Секреты исключены из репозитория

## v0.0.1

- Исходная версия бота, которая была опубликована на GitHub до рефакторинга
- Базовые команды `/newclient`, `/revoke`, `/show`
- Минимальный функционал для выдачи WireGuard-конфигов
